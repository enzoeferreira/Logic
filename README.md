### - Operadores

    + -> Adição
    - -> Subtração
    * -> Multiplicação
    / -> Divisão
    \ -> Divisão Inteira
    ^ -> Potenciação
    % ou mod -> Resto da divisão

    >  -> Maior
    <  -> Menor
    >= -> Maior ou igual
    <= -> Menor ou igual
    =  -> Igual
    <> -> Diferente

    e
    ou
    nao

### - Declaração de variável

    nome : tipo

Tipos de variável:

    inteiro
    real
    caractere
    logico

### - Declaração de arrays

    nome: vetor [a..b] de tipo          // Vetor
    nome: vetor [a..b, c..d] de tipo    // Matriz

Tipos de arrays:

    inteiro
    real
    caractere

### - Atribuição de valor

    x <- valor
    nome[pos] <- valor
    nome[linha, coluna] <- valor
